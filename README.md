
# Food Delivery App 🍔

## About the Project
This is a food delivery application built using Java and Spring Boot.

## Features
- User login & registration
- Browse restaurants
- Add to cart
- Payment integration

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML, CSS

## How to Run
1. Clone repository
2. Open in IntelliJ/Eclipse
3. Run main file
